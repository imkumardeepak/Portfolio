## Portfolio — Built With

This portfolio site is built with a modern, production-ready frontend stack:

| Technology | Purpose |
|---|---|
| React 19 + TypeScript | UI framework |
| Vite 7 | Build tool |
| Tailwind CSS 4 | Styling |
| Framer Motion | Animations |
| shadcn/ui + Radix UI | Component library |
| Canvas API | 3D animated globe |
| Vercel | Deployment |

### Run Locally

```bash
# Clone the repository
git clone https://github.com/imkumardeepak/portfolio.git
cd portfolio

# Install dependencies
cd artifacts/portfolio
npm install

# Start the development server
npm run dev
```

### Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/imkumardeepak/portfolio)

The `vercel.json` at the root handles all deployment configuration automatically. Just connect the repo and deploy.
